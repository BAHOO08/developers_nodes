## Git
### Поиск коммитов

#### Поиск коммитов по тексту
```bash
git log -S 'some text'
```

#### Поиск коммитов по тексту из коммита
```bash
git log --grep 'some text'
```

### Слияние коммитов
Перестраиваем последние 3 коммита
```bash
git rebase -i HEAD~3
```
В открытом окне vim пишем squash для тех коммитов, которые хотим оставить
жмём wq и затираем неинтерисующие сообщения с коммитами.
